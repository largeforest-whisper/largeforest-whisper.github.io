Be–Do–Have → Identity Model ▶︎ Integration

行動を変える前に、
自分の構造を整理する。

Be–Do–Have → Identity Modelは、 自分の在り方、行動、得ているもの、社会との接続を整理し、 分断された要素を統合するためのモデルです。

なぜ、行動だけ変えても続かないのか

やることを増やしても、自分の状態や価値観とズレていると、 行動は継続しにくくなります。 まず必要なのは、行動量ではなく構造の整理です。

Be / Do / Have / Identity

Beは在り方。Doは行動。Haveは得ているもの。 Identityは、それらが社会や他者とどう接続されているかを示します。 この4つを分けて見ることで、自分の現在地が整理されます。

Integration

統合とは、ただまとめることではありません。 バラバラに見えていた価値観、行動、成果、役割をつなぎ直し、 自分の方向性として再構築することです。
